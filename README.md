# Sky-Blue Community Platform

A sophisticated, real-time-inspired community platform for sharing information.

## ✨ Features
- **Modern UI:** Clean white and sky blue aesthetic using OKLCH color space.
- **Web Components:** Built with vanilla JavaScript Web Components and Shadow DOM for modularity.
- **Welcome Screen:** A polished landing experience for new users.
- **Category Navigation:** Easy browsing through various information sharing rooms.
- **Feedback Integration:** Seamless feedback submission via Formspree.

## 🚀 Deployment
This project is deployed using GitHub Pages. 
Visit the live site here: [https://100jwang.github.io/product_2/](https://100jwang.github.io/product_2/)

## 🛠 Tech Stack
- HTML5 / CSS3 (Baseline features like Cascade Layers, :has selector)
- Vanilla JavaScript (ES Modules, Web Components)
- Formspree (Feedback management)
